As credenciais da API permitem que sistemas externos acessem os recursos habilitados na sua conta Lazy Data, como consultas, validações, enriquecimentos, monitoramentos, armazenamento e histórico.

<Warning>
  As credenciais devem ser gerenciadas apenas pela conta principal. Subcontas não possuem acesso direto à API.
</Warning>

## Como criar uma credencial

<Steps>
  <Step title="Acesse o painel">
    Entre na sua conta Lazy Data e acesse o menu **Integrações \> API**.
  </Step>
  <Step title="Clique em Nova credencial">
    Na tela de API, clique em **Nova credencial** para abrir o formulário de criação.
  </Step>
  <Step title="Informe um nome de identificação">
    Defina um nome claro para a credencial, por exemplo: `Integração ERP`, `Sistema interno` ou `Ambiente de produção`.
  </Step>
  <Step title="Selecione os escopos permitidos">
    Escolha quais recursos essa credencial poderá acessar.

    Exemplos de escopos disponíveis:

    - **Consulta**
    - **Monitoramento**
    - **Enriquecimento**
    - **Armazenamento**
    - **Histórico**
  </Step>
  <Step title="Configure restrição por IP, se necessário">
    A restrição por IP é opcional. Quando habilitada, somente as origens informadas poderão usar a credencial.

    O campo aceita:

    - IP individual: `203.0.113.10`
    - Lista de IPs, um por linha
    - CIDR: `203.0.113.0/24`
    - Intervalo: `203.0.113.10-203.0.113.30`
  </Step>
  <Step title="Crie a chave">
    Clique em **Criar chave**. Após a criação, a chave de API será exibida no modal para cópia.
  </Step>
</Steps>
