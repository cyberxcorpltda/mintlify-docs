Cria um monitoramento de pessoa física para um CPF e uma ou mais fontes.

Cada fonte recebe sua própria frequência de checagem.

## Corpo da requisição

<ParamField body="name" type="string" required>
  Nome da pessoa ou nome interno do monitoramento.
</ParamField>

<ParamField body="tax_id" type="string" required>
  CPF da pessoa, com ou sem pontuação.
</ParamField>

<ParamField body="sources" type="array<object>" required>
  Fontes que serão monitoradas.
</ParamField>

<ParamField body="notification_email" type="string" required>
  E-mail que receberá os alertas.
</ParamField>

## Estrutura de `sources[]`

| Campo | Tipo | Descrição |
| --- | --- | --- |
| `source_id` | `string` | ID da fonte retornada em [Configuração](/api-reference/monitoring/person/config). |
| `frequency_id` | `string` | ID da frequência escolhida. |

## Exemplo de corpo

```json
{
  "name": "PESSOA EXEMPLO DOS SANTOS",
  "tax_id": "00000000000",
  "sources": [
    {
      "source_id": "person_processes",
      "frequency_id": "daily"
    }
  ],
  "notification_email": "alertas@example.com"
}
```

## Fluxo recomendado

1. Consulte [Configuração](/api-reference/monitoring/person/config).
2. Valide o CPF em [Validar pessoa física](/api-reference/monitoring/person/validate).
3. Crie o monitoramento com as fontes desejadas.
4. Acompanhe a checagem inicial em [Detalhar monitoramento](/api-reference/monitoring/detail).
