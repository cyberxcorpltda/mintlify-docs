Valida se o CNPJ informado pode ser monitorado.

<ResponseExample>

```json 200
{
  "code": 200,
  "message": "Pessoa jurídica validada para monitoramento.",
  "result": {
    "valid": true
  }
}
```

```json 400
{
  "code": 42013,
  "message": "CNPJ inválido."
}
```

</ResponseExample>

## Corpo da requisição

<ParamField body="name" type="string" required>
  Razão social ou identificador interno para validação.
</ParamField>

<ParamField body="tax_id" type="string" required placeholder="00000000000000">
  CNPJ da empresa, com ou sem pontuação.
</ParamField>

## Exemplo de corpo

```json
{
  "name": "EMPRESA EXEMPLO LTDA",
  "tax_id": "00000000000000"
}
```
