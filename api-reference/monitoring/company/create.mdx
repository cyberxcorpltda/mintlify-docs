Cria um monitoramento de pessoa jurídica para um CNPJ e uma ou mais fontes.

Cada fonte recebe sua própria frequência de checagem.

## Corpo da requisição

<ParamField body="name" type="string" required>
  Razão social ou nome interno do monitoramento.
</ParamField>

<ParamField body="tax_id" type="string" required>
  CNPJ da empresa, com ou sem pontuação.
</ParamField>

<ParamField body="sources" type="array<object>" required>
  Fontes que serão monitoradas.
</ParamField>

<ParamField body="notification_email" type="string" required>
  E-mail que receberá os alertas.
</ParamField>

## Estrutura de `sources[]`

| Campo | Tipo | Descrição |
| --- | --- | --- |
| `source_id` | `string` | ID da fonte retornada em [Configuração](/api-reference/monitoring/company/config). |
| `frequency_id` | `string` | ID da frequência escolhida. |

## Exemplo de corpo

```json
{
  "name": "EMPRESA EXEMPLO LTDA",
  "tax_id": "00000000000000",
  "sources": [
    {
      "source_id": "company_processes",
      "frequency_id": "daily"
    }
  ],
  "notification_email": "alertas@example.com"
}
```

## Fluxo recomendado

1. Consulte [Configuração](/api-reference/monitoring/company/config).
2. Valide o CNPJ em [Validar pessoa jurídica](/api-reference/monitoring/company/validate).
3. Crie o monitoramento com as fontes desejadas.
4. Acompanhe a checagem inicial em [Detalhar monitoramento](/api-reference/monitoring/detail).
