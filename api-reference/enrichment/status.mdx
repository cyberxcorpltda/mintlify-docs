Retorna os detalhes de um enriquecimento específico.

Use esta rota após o upload para verificar se o arquivo já foi analisado, durante a execução para acompanhar progresso, e após a conclusão para identificar os arquivos de saída disponíveis.

## Campos principais

| Campo | Descrição |
| --- | --- |
| `status` | Status atual do enriquecimento. |
| `file` | Metadados do arquivo enviado. |
| `progress` | Progresso de upload, análise ou execução. |
| `analysis` | Colunas identificadas, amostras e sugestão de coluna de documento. |
| `configuration` | Coluna e datasets configurados para execução. |
| `cost` | Valor estimado, reservado, consumido, estornado e preços por dataset. |
| `outputs` | Arquivos de saída disponíveis após o processamento. |

## Quando consultar

- Após o upload, consulte até `status` chegar em `waiting_configuration`.
- Durante execução, consulte até `status` chegar em `completed`, `completed_with_errors` ou `failed`.
- Após conclusão, leia `outputs` para saber quais arquivos podem ser baixados.
