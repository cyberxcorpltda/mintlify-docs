Gera um link temporário para baixar um arquivo de saída do enriquecimento.

Use esta rota após o status chegar em `completed` ou `completed_with_errors`.

<ResponseExample>

```json 200
{
  "code": 200,
  "message": "Link de download gerado.",
  "result": {
    "url": "https://s3.amazonaws.com/exemplo-assinado",
    "file_name": "registros_enriquecidos.csv",
    "output_type": "success"
  }
}
```

```json 404
{
  "code": 404,
  "message": "Arquivo do enriquecimento não localizado."
}
```

</ResponseExample>

## Parâmetros

<ParamField path="enrichment_id" type="string" required>
  ID do enriquecimento.
</ParamField>

<ParamField path="output_type" type="string" required placeholder="success">
  Tipo de arquivo de saída.
</ParamField>

Valores aceitos para `output_type`:

| Valor | Arquivo | Descrição |
| --- | --- | --- |
| `success` | `registros_enriquecidos.csv` | Registros enriquecidos com sucesso. |
| `not_found` | `nao_encontrados.csv` | Registros sem retorno nos datasets selecionados. |
| `failed` | `falhas.csv` | Registros com falha de processamento. |
| `invalid` | `documentos_invalidos.csv` | Registros ignorados por documento inválido. |

## Resposta

<ResponseField name="code" type="integer" required>
  Código da resposta.
</ResponseField>

<ResponseField name="message" type="string" required>
  Mensagem descritiva da operação.
</ResponseField>

<ResponseField name="result" type="object" required>
  Dados do link temporário de download.
</ResponseField>

## Estrutura de `result`

| Campo | Tipo | Descrição |
| --- | --- | --- |
| `url` | `string` | Link temporário para baixar o arquivo. |
| `file_name` | `string` | Nome sugerido para salvar o arquivo. |
| `output_type` | `string` | Tipo de saída solicitado. |

<Warning>
  O link retornado é temporário. Se expirar, gere um novo link chamando este endpoint novamente.
</Warning>

## Como saber quais arquivos existem

Consulte o enriquecimento em [Consultar status](/api-reference/enrichment/status) e verifique `outputs`.

Exemplo:

```json
{
  "outputs": {
    "success": {
      "label": "Registros enriquecidos",
      "rows": 120,
      "size": 45000,
      "available": true,
      "created_at": "2026-06-22T12:30:00+00:00"
    },
    "not_found": {
      "label": "Não encontrados",
      "rows": 0,
      "size": 0,
      "available": false,
      "created_at": null
    }
  }
}
```

Solicite download apenas para saídas com `available: true` e `rows` maior que `0`.
